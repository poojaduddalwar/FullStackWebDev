Pointers for block

  1. width of block level element is 100% of veiwport width
  2. height of a block level element isn't set by default
  3. height & width can be explicitly set using css

Pointers for inline

  1. width of block level element is dependent on the content
  2. height of a inline element isn't set by default
  3. height & width can not be explicitly set using css
  4. we cannot add div inside span or span cannot wrap div inside it
